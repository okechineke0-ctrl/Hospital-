## 🚀 To Connect Real Backend Later

### Step 1: Deploy Backend
- Follow DEPLOYMENT.md for Railway setup
- Get your backend URL (e.g., `https://hospital-production-xxxx.railway.app`)

### Step 2: Update Environment Variable
Add to Vercel environment variables:
```
VITE_API_URL=https://hospital-production-xxxx.railway.app/api
```

### Step 3: Redeploy
Push to GitHub, Vercel auto-updates!
